# Depends

### Install lua5.3

```sh
sudo apt install lua5.3 lua5.3-dev
```

### Install Clang-format

```sh
sudo apt install clang-format
```

### Install CMake

```sh
sudo apt install cmake
```

### Install gcc & g++

```sh
sudo apt install gcc g++
```

### Install pip3

```sh
sudo apt install python3-pip
```

### Install gdbgui

```sh
sudo pip3 install gdbgui --upgrade
```
